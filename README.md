# 🚀 Social Media Strategy Planner (OE16)

> AI-powered tool that generates complete social media strategies for content creators across different platforms.

---

## 📌 Overview

**Social Media Strategy Planner** is an intelligent system that helps creators and brands plan their content efficiently.

It uses a **multi-agent pipeline** to:

* Research audience
* Generate strategy
* Plan content
* Schedule posts
* Evaluate performance

All in one place ⚡

---

## 🎯 Problem It Solves

Creating consistent and effective social media content is difficult. This tool:

* ❌ Removes guesswork
* ✅ Automates planning
* 📈 Improves engagement strategy

---

## 🧠 How It Works

The system follows a structured **AI Agent Pipeline**:

1. **Strategy Planner** → Generates overall content strategy
2. **Audience Researcher** → Identifies target audience
3. **Content Planner** → Creates post ideas
4. **Scheduler** → Organizes posting timeline
5. **Judge & Evaluator** → Scores and improves the strategy

---

## ✨ Features

* 🔥 AI-powered multi-step strategy generation
* 🎯 Platform-specific planning (e.g., Instagram)
* 📊 Strategy scoring & evaluation system
* 📅 7-day content calendar generation
* ⚡ Fast execution pipeline
* 🧩 Modular agent-based architecture

---

## 🛠️ Tech Stack

* **Language:** Python 🐍
* **AI Model:** LLaMA-based model (via Groq API)
* **Search Integration:** Tavily Search

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/24jungharehm-dev/OE16.git
cd OE16
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Set environment variables

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key
TAVILY_API_KEY=your_api_key
```

### 4️⃣ Run the project

```bash
python app.py
```

---

## 🚀 Usage

1. Enter your **brand/product**
2. Select a **platform (e.g., Instagram)**
3. Click **Generate Strategy**
4. Get:

   * Content ideas
   * Audience insights
   * Posting schedule
   * Performance score
